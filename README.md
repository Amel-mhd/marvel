# Marvel Universe 🦸

Application web fullstack permettant d'explorer les personnages et comics Marvel.

## Fonctionnalités

- Liste des personnages Marvel avec recherche et pagination
- Liste des comics Marvel avec recherche et pagination
- Page détail d'un personnage avec ses comics
- Système de favoris sauvegardés en localStorage
- Composants réutilisables — Card, ButtonPrimary, ButtonSecondary, SearchInput
- Zéro style inline — CSS séparé par composant
- Requêtes centralisées dans api.js

## Technologies

### Frontend
- React
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express
- Axios

## Lancer le projet

### Frontend
\`\`\`bash
npm install
npm start
\`\`\`

### Backend
\`\`\`bash
cd marvel-backend
npm install
node server.js
\`\`\`
